# fitbit-2048

An unofficial Fitbit Versa 4 and Sense 2 clone of 2048. Swipe the board to combine tiles, and try to achieve the ever-elusive 2048! Keep track of your current score on top and your best score on the bottom! Long press anywhere to pause the game and open the menu!

## Installation

For development, clone the repo and refer to my unofficial guide [fitbit-versa4-sense2-sdk](https://github.com/yeohongred/fitbit-versa4-sense2-sdk) on how to build and install the app locally.
